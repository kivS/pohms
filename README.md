Pohms - resistor or not resistor detector


## Installation

- Install Poetry

- Install dependencies 

  ```bash
  poetry install --develop .
  ```

- Spawn poetry shell
  ```bash
  poetry shell
  ```
  
- Run `pohms`

  ```bash
  pohms
  ```



## Explore

- Run `ipython`
  ```bash
  poetry run ipython
  ```
