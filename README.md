Pohms - resistor or not resistor detector


## Installation

- Install Poetry

- Install dependencies 

  ```bash
  poetry install --develop .
  ```

- Spawn poetry shell
  ```bash
  poetry shell
  ```
  
- Run `pohms`

  ```bash
  pohms
  ```



## Explore

- Run empty `ipython`
  ```bash
  poetry run ipython
  ```

- Run `ipython` with loaded info
  ```bash
  poetry run ipython -i explore.py 
  ```
